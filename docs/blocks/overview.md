# Обзор блоков

Polar содержит 9 категорий блоков, аналогичных Scratch.

## Категории

| Категория | Цвет | Описание |
|-----------|------|----------|
| [События](events.md) | 🟡 Жёлтый | Точка входа в программу |
| [Движение](motion.md) | 🔵 Синий | Перемещение спрайтов |
| [Внешний вид](looks.md) | 🟣 Фиолетовый | Речь, видимость, костюмы |
| [Управление](control.md) | 🟠 Оранжевый | Циклы, условия, ожидание |
| [Датчики](sensing.md) | 🔷 Голубой | Ввод, столкновения, таймер |
| [Операторы](operators.md) | 🟢 Зелёный | Математика, логика |
| [Переменные](variables.md) | 🔴 Тёмно-оранжевый | Хранение данных |
| [Перо](pen.md) | 🟢 Зелёный | Рисование на сцене |
| [Значения](literals.md) | 🔵 Голубой | Числа, текст |

## Типы блоков

- **Стековые** (Statement) — выполняются по порядку, имеют вход/выход для соединения
- **Шапочные** (Hat) — точка входа, не соединяются с предыдущими блоками
- **Репортёрные** (Reporter) — возвращают значение, вставляются в входы других блоков
- **Булевы** (Boolean) — возвращают true/false, вставляются в условия
